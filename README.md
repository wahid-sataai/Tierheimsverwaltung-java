# Tierheimverwaltung (Java)

Desktop-Anwendung zur Verwaltung eines Tierheims.

## Beschreibung
Dieses Projekt ist eine Java-Swing-Desktopanwendung, die im Rahmen eines IHK-Abschlussprojekts entwickelt wurde.
Die Anwendung unterstützt die Verwaltung von Tieren, Fütterungsplänen, Gehegen und Öffnungszeiten.

## Funktionen
- Tierverwaltung (CRUD-Funktionen)
- Verwaltung von Fütterungsplänen
- Gehegeverwaltung
- Verwaltung der Öffnungszeiten
- Grafische Benutzeroberfläche mit Java Swing
- MySQL-Datenbankanbindung
- Objektorientierte Programmierung (OOP)

## Technologien
- Java
- Java Swing
- Maven
- MySQL
- JDBC
- Git & GitHub

## Projektstruktur
- `Main.java` – Einstiegspunkt der Anwendung  
- `MainFrame.java` – Hauptfenster der GUI  
- `Database.java` – Datenbankverbindung  
- `TierheimService.java` – Geschäftslogik  
- GUI-Panels für Tiere, Fütterung und Öffnungszeiten

## Ausführen der Anwendung
1. Repository klonen
2. Projekt in IntelliJ IDEA öffnen
3. MySQL-Datenbank konfigurieren
4. `Main.java` ausführen
